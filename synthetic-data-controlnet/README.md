# ДЗ 2.5 — Синтетические данные: Stable Diffusion + ControlNet

Аугментация редких классов синтетикой. Берём датасет из ДЗ2, находим редкие
классы, генерируем для них новые изображения через **Stable Diffusion 1.5 +
ControlNet (Canny)**, обучаем классификатор (ResNet18) **без** и **с** синтетикой
и сравниваем (ablation).

## Зачем ControlNet, а не просто SD
Чистый SD по тексту даёт случайные картинки. ControlNet с Canny берёт контур
реального объекта и держит его форму, меняя только текстуру/цвет/фон. Так
синтетика остаётся валидными примерами класса, но добавляет разнообразие.

## Установка
```bash
pip install -r requirements.txt
```
Нужен GPU с ~6+ ГБ VRAM (Colab T4 ок). Запуск из папки `src/`.
Сначала должен быть выполнен **ДЗ2** (нужны его данные `data/train`).

## Что и откуда скачивается
- **Веса Stable Diffusion 1.5** — `stable-diffusion-v1-5/stable-diffusion-v1-5`
  (официальное зеркало; старый `runwayml/stable-diffusion-v1-5` удалён с
  Hugging Face в августе 2024). Запасной вариант: `sd-legacy/stable-diffusion-v1-5`.
- **Веса ControlNet (Canny)** — `lllyasviel/sd-controlnet-canny`.
- Оба тянутся автоматически при первом запуске `generate_synthetic.py`.
- **Данные** — берутся из ДЗ2 (кропы объектов), отдельно качать не нужно.

## Запуск по шагам
```bash
cd src

# 1) датасет классификации из COCO-subset (кропы по GT-боксам)
#    в конце печатает число примеров по классам -> видно редкие
python make_classification_data.py

# 2) генерация синтетики для редких классов
#    (внутри список ["umbrella", "clock"] — поправьте по выводу шага 1)
python generate_synthetic.py

# 4-5) обучение без и с синтетикой + таблица сравнения
python ablation.py

# либо по отдельности:
python train_classifier.py
python train_classifier.py --use-synthetic
```

## Параметры
| Что              | Значение                                   |
| ---------------- | ------------------------------------------ |
| Генерация        | SD 1.5 + ControlNet Canny, 25 шагов, UniPC |
| Синтетики/картинку | 3                                        |
| Классификатор    | ResNet18 (предобучен на ImageNet)          |
| Эпохи / batch    | 8 / 32, Adam lr=1e-4, вход 128×128         |
| Split            | 80% train / 20% val (seed=0)               |

## Структура
```
src/
  make_classification_data.py  # шаг 1: кропы -> clf_data/<класс>/
  generate_synthetic.py        # шаг 2: SD+ControlNet -> synthetic/<класс>/
  train_classifier.py          # шаг 4: ResNet18, флаг --use-synthetic
  ablation.py                  # шаг 5: оба прогона + таблица
clf_data/                      # реальные кропы (после шага 1)
synthetic/                     # синтетика (после шага 2)
```

## Наблюдения (заполнить своими)

- вывод: даёт ли синтетика прирост и на каких классах сильнее.
